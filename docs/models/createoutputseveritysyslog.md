# CreateOutputSeveritySyslog

Default value for message severity. Will be overwritten by value of __severity if set. Defaults to notice.


## Values

| Name        | Value       |
| ----------- | ----------- |
| `EMERGENCY` | 0           |
| `ALERT`     | 1           |
| `CRITICAL`  | 2           |
| `ERROR`     | 3           |
| `WARNING`   | 4           |
| `NOTICE`    | 5           |
| `INFO`      | 6           |
| `DEBUG`     | 7           |
