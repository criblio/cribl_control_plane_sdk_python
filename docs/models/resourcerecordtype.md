# ResourceRecordType

The DNS record type (RR) to return. Defaults to 'A'.


## Values

| Name    | Value   |
| ------- | ------- |
| `A`     | A       |
| `AAAA`  | AAAA    |
| `ANY`   | ANY     |
| `CNAME` | CNAME   |
| `MX`    | MX      |
| `NAPTR` | NAPTR   |
| `NS`    | NS      |
| `PTR`   | PTR     |
| `SOA`   | SOA     |
| `SRV`   | SRV     |
| `TXT`   | TXT     |
